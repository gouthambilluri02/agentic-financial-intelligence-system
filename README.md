# Agentic Financial Intelligence System

An enterprise-style AI financial intelligence platform that uses multi-agent workflows, RAG, semantic search, and financial data analysis to generate market insights, portfolio summaries, and risk reports.

## Features

- Multi-agent AI workflow
- Financial document analysis
- RAG-powered question answering
- Semantic search using vector embeddings
- Portfolio and risk analysis
- FastAPI backend
- React dashboard frontend
- Docker-ready architecture

## Tech Stack

- Python
- FastAPI
- LangChain
- FAISS
- Pandas
- NumPy
- React
- Docker
- GitHub Actions
